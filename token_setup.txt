====================================
使用终端推送的步骤
====================================

第1步：创建GitHub Token
====================================

1. 访问：
   https://github.com/settings/tokens

2. 点击 "Generate new token" → "Generate new token (classic)"

3. 设置：
   Note: Spending Dashboard
   Expiration: No expiration (或选择一个日期)
   勾选权限：
   ☑ repo ( Full control of private repositories)

4. 点击 "Generate token"

5. 复制生成的token（只显示一次！）

====================================
第2步：使用Token推送
====================================

在终端执行：

cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"

git push https://<TOKEN>@github.com/shaohuayangLLM/spending-dashboard2025.git main

注意：将 <TOKEN> 替换为刚才复制的token
