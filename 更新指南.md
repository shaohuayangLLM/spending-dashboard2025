# 网站更新指南

## 🚀 三种更新方法

### 方法1：GitHub网页上传（最简单）⭐推荐

**适合：** 偶尔更新、少量修改

#### 步骤：
1. 修改本地文件（index.html、PDF等）
2. 访问：https://github.com/shaohuayangLLM/spending-dashboard2025
3. 点击右上角 "Add file" → "Upload files"
4. 拖拽文件到页面
5. 填写提交信息
6. 点击 "Commit changes"
7. 等待1-3分钟自动部署

**优点：** 不需要命令行，操作简单
**缺点：** 每次只能上传文件，不能修改文件内容

---

### 方法2：使用更新脚本（快捷）⭐推荐

**适合：** 已有Git仓库，快速更新

#### 使用方法：
```bash
cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"
./update_and_deploy.sh
```

脚本会自动：
- ✅ 检查修改的文件
- ✅ 添加所有修改
- ✅ 提交更改
- ✅ 推送到GitHub
- ✅ 触发自动部署

---

### 方法3：Git命令行（专业）

**适合：** 频繁更新、需要版本控制

#### 基本命令：
```bash
# 进入目录
cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"

# 查看修改
git status

# 添加文件
git add index.html
# 或添加所有修改
git add .

# 提交
git commit -m "Update: 更新内容描述"

# 推送
git push
```

## 🆘 常见问题

### Q: 更新后看不到变化？
A: 清除浏览器缓存（Cmd+Shift+R），等待1-3分钟

### Q: 推送失败？
A: 检查网络连接，确认有写入权限

### Q: 想回退到之前的版本？
A: 访问GitHub仓库的Commits历史，可以恢复旧版本
