====================================
GitHub Pages 部署步骤
====================================

第1步：在GitHub上创建仓库
====================================

1. 打开浏览器，访问：
   https://github.com/new

2. 填写仓库信息：
   Repository name: spending-dashboard
   Description: 消费数据对比分析网页
   
   ✓ 选择 Public（公开）
   ☐ 不要勾选 "Add a README file"
   ☐ 不要勾选 "Add .gitignore"
   ☐ 不要勾选 "Choose a license"

3. 点击绿色按钮：Create repository

====================================
第2步：复制仓库地址
====================================

创建后会显示你的仓库地址，类似：
https://github.com/shaohuayangLLM/spending-dashboard.git

====================================
第3步：告诉我"第1步完成"
====================================

完成后，回复"第1步完成"，我会帮您推送代码！
