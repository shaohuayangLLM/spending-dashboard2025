# 消费数据网页部署指南

## 方案一：GitHub Pages（推荐，完全免费）

### 步骤1：创建GitHub仓库
1. 访问 https://github.com/new
2. 仓库名：`spending-dashboard`（或其他你喜欢的名字）
3. 设置为 Public（公开）
4. 不要勾选 "Add a README file"
5. 点击 "Create repository"

### 步骤2：上传文件到GitHub
```bash
# 进入你的数据文件夹
cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"

# 初始化Git仓库
git init

# 添加文件
git add spending_comparison.html

# 提交
git commit -m "Initial commit: 消费数据对比网页"

# 添加远程仓库（替换YOUR_USERNAME为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/spending-dashboard.git

# 推送到GitHub
git branch -M main
git push -u origin main
```

### 步骤3：启用GitHub Pages
1. 在你的GitHub仓库页面，点击 Settings (设置)
2. 左侧菜单找到 "Pages"
3. Source 选择：
   - Branch: `main`
   - Folder: `/root`
4. 点击 Save

### 步骤4：访问网页
等待1-2分钟后，你的网页将发布在：
```
https://YOUR_USERNAME.github.io/spending-dashboard/
```

## 方案二：Vercel（更简单，自动部署）

### 步骤1：安装Vercel CLI
```bash
npm install -g vercel
```

### 步骤2：登录并部署
```bash
cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"
vercel login
vercel --prod
```

### 步骤3：获取访问链接
部署完成后，Vercel会给你一个URL，例如：
```
https://spending-dashboard.vercel.app
```

## 方案三：Netlify（拖拽部署）

```
https://amazing-jones-123456.netlify.app
```

## 方案四：Cloudflare Pages（免费且快速）

### 步骤1：访问Cloudflare Pages
打开 https://dash.cloudflare.com/

### 步骤2：创建项目
1. 登录或注册Cloudflare账号
2. 选择 "Workers & Pages"
3. 点击 "Create application"
4. 选择 "Pages" 标签
5. 点击 "Upload assets"

### 步骤3：上传文件
1. 将 `spending_comparison.html` 重命名为 `index.html`
2. 上传文件
3. 部署完成

## 快速部署脚本（GitHub Pages）

保存以下脚本为 `deploy.sh`，然后运行：

```bash
#!/bin/bash

# 配置
GITHUB_USERNAME="你的GitHub用户名"
REPO_NAME="spending-dashboard"

# 进入目录
cd "/Users/ysh/Manual Library/ClaudeCode/Vibe Coding/消费数据"

# 初始化Git
git init
echo "node_modules/" > .gitignore
echo ".DS_Store" >> .gitignore

# 添加文件
git add spending_comparison.html
git add 消费数据对比报告2023-2026.pdf
git add spending_data_full.json

# 提交
git commit -m "Add spending dashboard"

# 添加远程仓库
git remote add origin "https://github.com/${GITHUB_USERNAME}/${REPO_NAME}.git"

# 推送
git branch -M main
git push -u origin main

echo "部署完成！"
echo "请访问："
echo "https://github.com/${GITHUB_USERNAME}/${REPO_NAME}/settings/pages"
echo "启用GitHub Pages后，网页将在以下地址可访问："
echo "https://${GITHUB_USERNAME}.github.io/${REPO_NAME}/"
```

运行脚本：
```bash
chmod +x deploy.sh
./deploy.sh
```

## 数据隐私提醒

⚠️ **重要提示：**
- 如果部署到公网，任何人都可以访问你的消费数据
- 如果数据包含敏感信息，建议：
  1. 使用密码保护
  2. 或只在本地网络/私有服务器部署
  3. 或对敏感数据进行脱敏处理

## 自定义域名（可选）

如果你有自己的域名，可以添加到GitHub Pages：

1. 在仓库的 Settings > Pages 中
2. 点击 "Add a domain"
3. 输入你的域名（如：finance.yourdomain.com）
4. 按照提示配置DNS

---

## 更新网页

更新网页后，只需运行：

```bash
git add .
git commit -m "Update data"
git push
```

GitHub Pages会自动重新部署！
